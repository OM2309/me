# anurag-uat.vercel.app
 
Personal portfolio — built with **Next.js 14 (App Router)**, **TypeScript**, **Tailwind CSS**, **Drizzle ORM**, and deployed on **Vercel**.
 
**Live →** [anurag-uat.vercel.app](https://anurag-uat.vercel.app)

| Layer | Technologies |

## Features
 
- App Router with React Server Components
- Dark / Light mode with system preference detection
- Smooth scroll animations via Framer Motion
- GitHub contributions graph
- Blog with MDX support
- Contact form powered by Resend
- SEO-optimised with OG images
- Vercel Analytics & Speed Insights

## Getting Started
 
```bash
git clone https://github.com/OM2309/me.git
cd me
pnpm install
pnpm dev
```
 
### Useful Scripts
 
```bash
pnpm dev          # Start dev server
pnpm build        # Production build
pnpm lint         # Run ESLint
pnpm db:push      # Push schema to DB
pnpm db:generate  # Generate Drizzle types
```
